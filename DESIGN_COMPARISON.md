| 特性 | 旧设计 | 新设计 | 改进 |

## 🎨 视觉细节对比

### 背景效果

**旧设计**:
- 纯色背景
- 轻微透明度
- 无模糊效果

**新设计**:
- 玻璃态背景
- 92% 不透明度
- 16px 毛玻璃模糊
- 180% 饱和度增强

### 阴影系统

**旧设计**:
```css
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15);
```
单层阴影，较平面

**新设计**:
```css
/* 深色主题 */
box-shadow: 
  0 8px 32px rgba(0, 0, 0, 0.4),
  0 2px 8px rgba(0, 0, 0, 0.2);

/* 浅色主题 */
box-shadow: 
  0 8px 32px rgba(0, 0, 0, 0.1),
  0 2px 8px rgba(0, 0, 0, 0.06);
```
双层阴影，更有深度

## 🎬 动画对比

### 进入动画

**旧设计**:
```
淡入: opacity 0 → 1
时长: 200ms
缓动: linear
```

**新设计**:
```
淡入 + 滑入:
  opacity: 0 → 1
  transform: translateY(-20px) → translateY(0)
时长: 400ms
缓动: cubic-bezier(0.34, 1.56, 0.64, 1) (弹性)
```

### 进度更新

**旧设计**:
```
width: 300ms ease
```

**新设计**:
```
width: 350ms cubic-bezier(0.4, 0, 0.2, 1)
```

## 🔧 技术对比

### 实现方式

**旧设计**:
```typescript
// 使用 Obsidian Notice API
this.notice = new Notice('', 0);
this.notice.setMessage(html);
```

**新设计**:
```typescript
// 自定义 DOM 元素
this.container = document.body.createDiv('linguasync-toast');
// 完全控制样式和动画
```

### 性能

**旧设计**:
- 依赖 Notice API
- 有限的样式控制

**新设计**:
- 直接 DOM 操作
- CSS3 硬件加速动画
- 更流畅的性能
