# 🚀 快速开始指南 - OB-English-Learner TTS

## 📖 5分钟上手所有新功能

---

## 第一步：重启 Obsidian

```
按 Ctrl + R（或 Cmd + R on Mac）
```

等待插件重新加载...

---

## 第二步：基础播放测试

### 1. **选中一段文本**
```
在任何笔记中选中一段包含多个句子的文本
例如：
"This is sentence one. This is sentence two. This is sentence three."
```

### 2. **开始播放**
```
方式 1：按 Ctrl+Space（推荐）
方式 2：点击工具栏的 ▶ 按钮
```

### 3. **观察新功能**
```
✅ 通知："🔊 Playing 3 sentences"
✅ 当前句子紫色高亮
✅ 工具栏底部出现蓝色进度条
✅ 音频可视化器跳动
✅ 自动滚动到当前句子
```

---

## 第三步：测试键盘快捷键

### ⌨️ 播放控制
```
Ctrl + Space    开始播放（或暂停/恢复）
Esc             停止播放
```

**测试**：
1. 播放时按 `Ctrl+Space` → 看到 "⏸ Paused"
2. 再按一次 `Ctrl+Space` → 看到 "▶ Resumed"
3. 按 `Esc` → 看到 "⏹ Stopped"

---

### ⏮⏭ 句子导航
```
Ctrl + →        下一句
Ctrl + ←        上一句
```

**测试**：
1. 播放时按 `Ctrl+→` → 看到：
   - 通知："⏭ Next sentence"
   - 高亮跳到下一句
   - 进度条前进
2. 按 `Ctrl+←` → 看到：
   - 通知："⏮ Previous sentence"
   - 高亮回到上一句
   - 进度条后退

---

### ⚡ 速度调整
```
Ctrl + ↑        加速（+0.1x）
Ctrl + ↓        减速（-0.1x）
```

**测试**：
1. 播放时按 `Ctrl+↑` 几次 → 看到：
   - 通知："⚡ Speed: 1.10x"
   - 通知："⚡ Speed: 1.20x"
   - 通知："⚡ Speed: 1.30x"
2. 按 `Ctrl+↓` → 看到：
   - 通知："⚡ Speed: 1.20x"
   - 音频速度立即变化

## ⚠️ 常见问题

### Q: 高亮没有显示？
```
A: 检查：
   1. 是否正在播放？（应该看到可视化器）
   2. 是否在正确的编辑器？（工具栏在哪个编辑器顶部）
   3. 重启 Obsidian (Ctrl+R)
```

### Q: 快捷键不工作？
```
A: 检查：
   1. 焦点是否在编辑器？（点击编辑器）
   2. 其他插件是否占用了快捷键？
   3. 查看设置 → 热键，确认快捷键已注册
```

### Q: 进度条不显示？
```
A: 检查：
   1. 是否正在播放？
   2. 是否有多个句子？（单句可能不明显）
   3. 查看浏览器开发者工具的控制台
```

### Q: 速度调整没反应？
```
A: 检查：
   1. 是否在播放状态？
   2. 观察通知消息，确认速度在变化
   3. 重新播放以应用新速度
```

