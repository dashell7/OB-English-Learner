# ✅ 已添加：回车发送按钮

根据您的要求，我已经在输入框内部添加了一个**回车发送按钮**。

### 🔄 变更说明
*   **位置**: 输入框内部右下角（悬浮设计，类似于 ChatGPT）。
*   **样式**:
    *   **默认**: 灰色半透明，不引人注目。
    *   **有内容**: 变为**高亮色 (Accent Color)**，提示可发送。
*   **功能**: 点击按钮可直接发送消息（等同于按下回车）。

### 🎨 界面效果
```
+------------------------------------+
| 输入框内容...                      |
|                                    |
|                                [➣] | <-- 发送按钮
+------------------------------------+
```

这种设计既保留了极简风格（不需要额外的底部栏），又提供了明确的视觉反馈和点击操作。

请重启 Obsidian (`Ctrl+R`) 查看效果。
